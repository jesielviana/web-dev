## Objetivos da aula - Conhecer um pouco da história da Internet e da Web; - Entender os conceitos da Internet e Web e diferenciá-los.
## Surgimento da Internet - A internet surgiu a partir de pesquisas militares no auge da Guerra Fria na década de 1960 [1]. - Inicialmente foi chamada de ARPANET, criada pela ARPA, sigla para Advanced Research Projects Agency [1].
## Surgimento da Web - World Wide Web (Web) foi criada pelo inglês Tim Berners-Lee no CERN [1]. - Era um sistema de gerenciamento de informações, no qual o texto poderia conter links e referências a outros trabalhos. Tim criou um servidor para publicar esse estilo de documento (chamado hipertexto), bem como um programa para lê-los, que ele havia chamado WorldWideWeb, lançado pela primeira vez em 1991.

# Web? ## Internet? ### Rede de Computadores?

Rede de Computadores

Rede de computadores é a interconexão entre computadores que permite a comunicação de dados entre si através de cabos ou sem fios [2].

Internet

  • Uma rede que interconecta dispositivos em escala global para a transferência de dados entre eles.
  • É uma rede de várias outras redes.

Web

  • Sistema de documentos em hipermídia que são interligados e executados por meio da Internet.

Internet x Web

A Internet é a infraestrutura, enquanto a Web é um serviço construído sob esta infraestrutura [2].
